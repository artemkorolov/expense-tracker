# Expense Tracker

A simple and efficient web application to manage your personal finances by tracking income and expenses.

## 🛠 Tech Stack

*   **JavaScript (ES6+)** — Core logic for managing transactions and state.
*   **Persistent Storage** — Using `localStorage` to save data between browser sessions.
*   **DOM Manipulation** — Dynamic updates of the balance, history, and transaction lists.
*   **CSS3** — Custom styling with a dark theme and responsive layout.
*   **HTML5** — Сlean document structure.

## 🌟 Key Features

*   **Balance Tracking** — Real-time calculation of total balance, income, and expenses.
*   **Transaction History** — A detailed list of all activities with the ability to delete entries.
*   **Data Persistence** — Transactions are stored locally, so they won't disappear after a page refresh.
*   **Dynamic UI** — Automated ID generation and instant DOM updates without page reloads.
